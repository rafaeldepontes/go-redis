# Go-Redis

Project built with [Gini](https://gini-webserver.up.railway.app/)

---

A minimal Redis-like server implemented in Go, created for learning purposes.

This project focuses on understanding how Redis works internally by rebuilding some of its core concepts such as networking, command parsing, and in-memory data storage.

---

## About

This implementation was built while following the guide:

- https://www.build-redis-from-scratch.dev/en/introduction

The goal was not to create a production-ready Redis replacement, but to explore the architecture and core mechanisms behind Redis by implementing them step by step in Go.

---

## Tech Stack

- Go

---

## Purpose

The main objectives of this project are:

- Understand how Redis handles TCP connections
- Learn how commands are parsed and executed
- Explore in-memory data structures
- Study the fundamentals of key–value databases

---

## Disclaimer

This project is a learning exercise and does **not** aim to fully replicate Redis behavior or performance.
